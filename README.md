# E-Commerce Product Page

An interactive sneakers product page built with vanilla HTML, CSS and JavaScript.

## Features
- Image gallery with previous/next navigation
- Thumbnail switching
- Lightbox overlay on image click
- Cart functionality with add/remove items
- Quantity counter with plus/minus buttons
- Responsive mobile design with hamburger menu
- Discount pricing display

## Live Demo
https://jairam13.github.io/sneakers-product-page/

## Built With
- HTML5
- CSS3
- Vanilla JavaScript
